Links in HTML5

Using external links

The tags <a> and <⁄a> creates a link to an external destination. It is recommended to use the attribute 'target="_blank"' to allow the original website to remain open.

The following websites can be accessed by clicking their link:

    Youtube
    Instagram
    Twitch
    Discord

Using internal links

Internal links redirects the user to another webpage within the original server and relates to the previous webpage. These links can be created using <a> and <⁄a> tags and the attribute 'target' should be set as '_self' allowing navegation in the website within the same browser tab.
