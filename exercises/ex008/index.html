Main Formating Elements

Bold / Highlighted

In this phrase, there can be bold words using <b> and <⁄b> tags (non semantic, hence, deprecated).

In comparison, in this phrase there are hightlighted words using <strong> and <⁄strong> tags (semantic).

Italic / Emphasis

Italic words are also possible using <i> and <⁄i> tags (non semantic, hence, deprecated).

In comparison, in this phrase there are emphasized words using <em> and <⁄em> tags (semantic).

Marked Text

There can be marked text using <mark> and <⁄mark> tags

Marked text color can be changed by using the attribute inside of tag (uses CSS)

Big text and small text

In this paragraph there are big sized words and small sized words. Although it works just fine, <big> is deprecated but <small> can still be used.

Reason: <big> just makes words have a greater size than the rest, giving it no meaning behind it (non semantic); <small>, on the other hand, can be used to create fine prints in some agreements, giving it a semantic meaning.

Deleted Text

Deleted text can be done by using <del> and <⁄del> tags. It is marked as deleted to indicate that it can be read but not considered.

Inserted Text

Inserted text emphasizes that it was added later on and can be achieved by using <ins> and <⁄ins> tags.

Using <u> and <⁄u> tags, text can also be marked as underlined, although it has no semantic value.

Superscript Text

Superscript text can be used in some mathematical terms such as in x6+3 (where 6 is superscript). It can be done by using <sup> and <⁄sup> tags.

Subscript Text

Some chemical formulas make use of subscript text such as H2O (where 2 is subscript). It can be done by using <sub> and <⁄sub> tags.
