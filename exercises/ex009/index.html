Handling Lists

Ordered Lists (ol)

An ordered list in HTML means that the items in that list will be ordered according to the selected type (1, A, a, I, i). They can have a starting point (using the 'start' attribute) or just start from the beginning.

  1. Wake up
  2. Brush teeth
  3. Have breakfast
  4. Learn a skill
  5. Have a break
  6. Lunch

To create an ordered list, we first indicate the type of list using the tags <ol> and <⁄ol>. The items in the list should be included inside these tags using <li> (since HTML5, the closing tag for 'li' is not required).

Unordered Lists (ul)

An unordered list in HTML does not order items by any numerical or alphabetical order: a simple marker is placed before the item name (The attribute 'type' can change the marker (disc, circle, square)). Therefore, the items' position in this list is irrelevant.

  • League of Legends
  • Genshin Impact
  • Elden Ring
  • Valorant
  • Sekiro: Shadows die twice
  • Paladins
  • Brawlhalla

To create an unordered list, we first indicate the type of list using the tags <ul> and <⁄ul>. The items in the list should be included inside these tags using <li> (since HTML5, the closing tag for 'li' is not required).
